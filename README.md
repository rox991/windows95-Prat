# Windows 95 Portfolio OS

A fully functional, interactive Windows 95-themed portfolio website built with pure HTML, CSS, and vanilla JavaScript.

## Features

### Core OS Features
- Windows 95 authentic UI with pixel-perfect design
- Startup and shutdown sequences
- Desktop with double-clickable icons
- Start menu with nested menus
- Taskbar with running applications
- Real-time clock
- Context menu (right-click on desktop)

### Window Management
- Draggable, resizable windows
- Minimize, maximize, close functionality
- Active window highlighting
- Z-index management (click to bring to front)
- Taskbar integration
- Window snapping (drag near edges)

### Portfolio Applications
1. **My Computer (About Me)** - Personal introduction and profile
2. **My Documents (Projects)** - Project showcase with interactive cards
3. **Control Panel (Skills)** - Skills visualization with animated progress bars
4. **Internet Explorer (Contact)** - Working contact form with validation
5. **Notepad (Bio)** - Editable notepad with auto-save
6. **Recycle Bin** - Fun Easter egg with visual effects

### Advanced Features
- CRT screen effect toggle (F11 key)
- Sound effects toggle
- Alt+Tab window cycling
- Keyboard shortcuts (Esc to close)
- Responsive design for mobile devices
- Error dialog system
- Tooltip hints
